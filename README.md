# 📊 Nifty 100 Stock Analysis & DCF Valuation Model

## 🔍 Overview
This project presents an interactive financial analysis platform combining stock price forecasting and intrinsic valuation techniques.

It includes:
- Nifty 100 stock analysis terminal
- Monte Carlo simulation for future price forecasting
- Discounted Cash Flow (DCF) valuation model

---

## ⚙️ Tools & Technologies
- R
- Shiny
- Financial Modeling
- Monte Carlo Simulation
- DCF Valuation

---

## 🚀 Live Applications
- Nifty 100 Terminal: https://aditya-aj.shinyapps.io/Desktop/
- DCF Model: https://aditya-aj.shinyapps.io/projectvally/

---

## 📊 Key Features
- Simulates multiple future price scenarios under uncertainty
- Helps evaluate risk and return trade-offs
- Supports strategic investment decision-making
- Provides intrinsic valuation using DCF approach

---

## 🧠 Strategic Perspective
This project focuses on bridging:
- Data analysis → Financial insights  
- Financial insights → Strategic decisions  

It is designed from the perspective of an analyst, strategist, and decision-maker.
